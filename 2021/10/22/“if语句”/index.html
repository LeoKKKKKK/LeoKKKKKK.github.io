“if语句”

Posted on 2021-10-22

if语句=1.一般形式：

if (表达式) {语句;}

表达式：
a,用非 0 值表示真，用 0 表示假;
b,if(flag) 相当于 if(1==flag)
c,浮点数无法与 0 比较，只能用近似的值比较；例: 1e-6 等于1x10的-6次方可以看成0来使用；

2.用于单分支选择结构;
3.如含有交叉关系，使用并列的if语句;

例1：输出两个整数中的最大值

二、if(…) {…} else {…}

一般形式：if(表达式) {语句1; } else { 语句2;}
用于双分支控制的条件语句;
用于非此即彼的关系时;
例2：输出两个整数中的最大值

例2关于对于非法字符的检查与处理

三、if(…) {…} else if(…) {…} … else if(…) {…} else {…}

一般形式：
if(表达式1) { 语句1;}
else if(表达式2) {语句2;}
…
else if(表达式n){语句n;}
else{语句n+1;}

用于多分支的控制的条件语句;

例3：判断字符

四、switch() {case …: …; case …: …; … default: …;}

一般形式：
switch(表达式)
{
case 常量1:语句1;
case 常量2:语句2;
…
case 常量n:语句n;
default: 语句n+1;
}
用于多路选择的语句;
switch语句相当于多个if-else语句;
(表达式)只能是char型或者int型;
case 后面至少要有一个空格，常量后面是冒号
(表达式)与 常量类型要保持一致;
记得记得记得 在需要跳出的时候，在语句后面加上break;

例4：简单的加减乘除计算
